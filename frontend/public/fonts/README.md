Coloca aquí los archivos WOFF2 de Hack Nerd Font Mono para servirlos como webfont.

Archivos sugeridos (mínimo):
- HackNerdFontMono-Regular.woff2

Opcionales (recomendado para consistencia):
- HackNerdFontMono-Bold.woff2
- HackNerdFontMono-Italic.woff2
- HackNerdFontMono-BoldItalic.woff2

Cómo obtener WOFF2:
- Desde el repositorio de Nerd Fonts (descargar Hack.zip) y convertir TTF a WOFF2 con herramientas como:
  - woff2: <https://github.com/google/woff2> (woff2_compress)
  - o un convertidor online confiable.

Con estos archivos, el navegador usará la fuente hospedada aquí. Si no están presentes,
la app intentará usar la fuente instalada en el sistema (local('Hack Nerd Font Mono')).
